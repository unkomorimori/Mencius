# 孟子

## P.33 1-王も、ただ仁義を曰わんのみ。何ぞ必ずしも利を曰わん。
http://www5f.biglobe.ne.jp/~mind/knowledge/classic/moushi001.html 
```
孟子、梁（りょう）の恵王（けいおう）に見ゆ（まみゆ）。

王（おう）曰く（いわく）、
叟（そう）、千里を遠しとせずに来たる。

亦（また）将（しょう）に以って吾が国を利（り）することあらんとするか。

孟子対えて（こたえて）曰く、王何ぞ必ずしも利を曰（い）わん、
亦（ただ）仁義あるのみ。

王は　何を以って吾が国を利せんと曰（い）い、
大夫（たいふ）は　何を以って吾が家（や）を利せんと曰い、
士（し）・庶人（しょじん）は　何を以って吾が身を利せんと曰いて、
上下（しょうか）交（こもごも）利を征らば（とらば）、
而ち（すなわち）国危うし。

万乗（ばんじょう）の国、その君（きみ）を弑（しい）する者は、
必ず千乗（せんじょう）の家（いえ）なり。
千乗の国、その君を弑する者は、必ず百乗の家なり。
万に千を取り、千に百を取るは、多からずと為さず。

然（しか）れども　苟くも（いやしくも）
義（ぎ）を後（あと）にして利を先（さき）にすることあらば、
奪わざれば厭かず（あかず）。

未だ仁にしてその親（おや）を遺つる（すつる）者はあらざるなり。
未だ義にしてその君（きみ）を後（あなど）る者はあらざるなり。

王も、ただ仁義を曰わんのみ。
何ぞ必ずしも利を曰わん。 
```

## P.36 2-賢者にして後能く此れを楽しむ
```
孟子　梁（りょう）の恵王（けいおう）に見（まみ）ゆ。

王（おう）沼（いけ）の上（ほとり）に立ち、
鴻鴈（こうがん）麋鹿（びろく）を顧（かえり）みて曰く、
賢者も亦た此れ（これ）を楽しむか。

孟子、対へて曰く、
★賢者にして後（のち）能（よ）く此れ（これ）を楽しむ。
不賢者は此れ（これ）有りと雖（いえど）も楽しまざるなり。

詩（し）に云ふ、
文王（ぶんのう）霊台（れいだい）を経始（けいし）す。
之を経り（はかり）　
之を営り（なわばり）、
庶民之を攻め（おさめ）、
日（ひ）ならず（幾日も経ず）して之を成せり。

経始（けいし）亟す（うながす）勿（な）きも、
庶民子（こ）のごとく来（きた）れり。

王（おう）霊囿（れいゆう）に在り、
麀鹿（ゆうろく）の伏（ふ）す攸（ところ）。

麀鹿（ゆうろく）濯濯（こくこく）たり。
白鳥鶴鶴（かくかく）たり。

王（おう）霊沼（れいしょう）に在り、
於牣（ここにみ）ちて魚（さかな）躍ると。

文王（ぶんのう）民（たみ）の力を以（もち）いて、
台（うてな）を為（おさ）め沼（いけ）を為（つく）りて、
民（たみ）之を歓楽び（よろこび）、
其の台（うてな）を謂（なずけ）て霊台（れいだい）と曰ひ、
其の沼（いけ）を謂（なずけ）て霊沼（れいしょう）と曰いて、
其の麋鹿（びろく）魚鼈（ぎょべつ）有るを楽しめり。

古（いにしえ）の人は民と與（とも）に楽しむ。
故に能（よ）く楽しめるなり。

湯誓（とうせい）に　民・傑王（けつおう）を日（ひ）に比（ひ）して曰く、
時（こ）の日　害（いつ）か喪（ほろ）ぶる、
予（われ）女（なんじ）と偕（とも）に亡（ほろ）びんと。

民（たみ）之と偕に亡びんと欲せば、
台池（だいち）鳥獣（ちょうじゅう）有りと雖も（いえども）、
豈（あに）能（よ）く独（ひと）り楽しまんや。
```
