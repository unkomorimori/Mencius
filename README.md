### 6. 惻隠の心は、仁の端なり
```
是れに由りて之を観れば、

惻隠の心無きは、人に非ざるなり。
羞悪の心無きは、人に非ざるなり。
辞譲（じじょう）の心無きは、人に非ざるなり。
是非の心無きは、人に非ざるなり。

惻隠の心は、仁の端（はじめ）なり。
羞悪の心は、義の端（はじめ）なり。
辞譲（じじょう）の心は、礼の端（はじめ）なり。
是非の心は、智の端（はじめ）なり。

```
